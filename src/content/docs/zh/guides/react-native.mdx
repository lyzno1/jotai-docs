Jotai 的原子可以在 React Native 应用中直接使用，完全无需任何修改。我们的目标是始终与 React Native 保持 100% 兼容。

### 持久化

关于持久化功能，React Native 特有的实现细节详见 [atomWithStorage 函数（utils 包）](../utilities/storage.mdx)。

### 性能

在应用中使用 Jotai 没有已知的额外开销。一些库会为了实际使用而在存储的数据上添加大量额外的属性和方法，但 Jotai 的做法不同——你始终在操作简单的数据，几乎没有可以进一步优化的空间。

Jotai 的原子化架构会鼓励你拆分逻辑和数据，让你能精确控制每一次渲染（[准确地说是 commit](https://legacy.reactjs.org/blog/2018/09/10/introducing-the-react-profiler.html#browsing-commits)），从而达到最佳性能。

始终记住：渲染必须快速，请将计算逻辑拆分到异步操作中。
