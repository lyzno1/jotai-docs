- 使用 atomFamily 和 localStorage 的 Todos 示例 [![Open in CodeSandbox](https://img.shields.io/badge/Open%20in-CodeSandbox-blue?style=flat-square&logo=codesandbox)](https://githubbox.com/pmndrs/jotai/tree/main/examples/todos_with_atomFamily)

  使用 atomFamily 和 localStorage 实现待办事项列表。你可以点击 `Save to localStorage` 将待办事项保存到 localStorage，然后删除待办事项，再点击 `Load from localStorage` 尝试恢复它们。
